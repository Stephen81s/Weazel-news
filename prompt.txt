🟥 1. FREECAM — SPÉCIFICATIONS FONCTIONNELLES
✔ 1.1 Accès configurable via fichier séparé
Créer un fichier séparé :
freecam_config.json (ou .lua selon le framework)

Il doit contenir :

Code
FreecamByJob = true/false
FreecamJobs = ["journaliste", "admin"]

FreecamByPlayer = true/false
FreecamWhitelist = ["license:xxxx", "steam:xxxx"]

FreecamEveryone = false

FreecamMaxDistance = 100
FreecamMinSpeed = 0.05
FreecamMaxSpeed = 20

POVEnabled = true
FiltersEnabled = true
✔ 1.2 Règles de la freecam
Aucun prop dans la main

Le joueur peut bouger librement

Le joueur peut faire des emotes

La caméra est indépendante

Distance max configurable (0–100 m)

Vitesse variable (0.05 → 20 m/s)

POV activables/désactivables

Filtres activables/désactivables

Audio basé sur le joueur

Logs détaillés pour chaque activation/désactivation
